# Editor de tablas de códigos

El editor de tablas de códigos te permite crear tablas de códigos para Digi3D.NET.

El archivo de configuración de Digi3D.NET o archivo de tablas \(llamado históricamente Digi.TAB\) especifica el conjunto de códigos con los que puede trabajar un usuario en Digi3D.NET y proporcionan a Digi3D.NET tanto información de representación de cada uno de los códigos como los parámetros de transformación de esos códigos a los distintos formatos de archivo de dibujo así como de guiones \(en cualquier lenguaje .NET\) que se pueden ejecutar cada vez que se almacena una geometría de un determinado código. Estos guiones pueden servir para comunicar errores al operador en caso de que la geometría no cumpla con un determinado criterio que no se ajuste al modelo semántico del proyecto para el cual se ha creado la tabla de códigos.

Una tabla de códigos está formada por siete secciones.

* [Paleta de colores](colores.md)
* Esquema de base de datos
* Estilos visuales
* Macroinstrucciones a ejecutar cuando se selecciona cualquier código
* Macroinstrucciones
* Reconocimiento de voz
* Códigos

 El editor de tablas de códigos muestra una pestaña por cada sección de una tabla de codigos.
