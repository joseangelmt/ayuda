# MOSTRAR\_STEREO

Activar/desactiva tanto la imagen raster cómo la superimposición.

## Parámetros

| Parámetro | Descripción |
| :--- | :--- |
| Sin parámetro | Alterna la visualización entre: Mostrar sólo raster/Mostrar sólo superimposición/Mostrar todo |
| 0 | Muestra sólo la superimposición |
| 1 | Muestra sólo la imagen raster |
| 2 | Muestra todo |
