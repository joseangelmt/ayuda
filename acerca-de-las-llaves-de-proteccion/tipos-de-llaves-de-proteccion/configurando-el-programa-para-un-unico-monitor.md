# Configurando el programa para un único monitor

Sigue los siguientes pasos para con figurar Digi3D.NET para un único monitor:

1. Ejecuta **Digi3D.NET**.
2. Aparecerá el cuadro de diálogo **Nuevo proyecto**.
3. Pulsa el botón **Cancelar** para cerrar el cuadro de diálogo **Nuevo proyecto**.
4. Selecciona la opción del menú **Herramientas/Configuración**.
5. Despliega la sección **Comunicación con el usuario**.
6. Cambia el valor del campo **Configuración para un único monitor** a **Si**.
7. Pulsa el botón **Aceptar** para aceptar el cuadro de diálogo **Configuración**.
8. Aparecerá un cuadro de diálogo indicándote que el cambio que has realizado requiere reiniciar Digi3D.NET. Pulsa el botón **Aceptar**.
9. Cierra **Digi3D.NET**.
10. Vuelve a ejecutar **Digi3D.NET**.
11. Aparecerá el cuadro de diálogo **Nuevo proyecto**.
12. Selecciona la pestaña **Sensores fotogramétricos**.
13. Pulsa el botón de los **tres puntos**. Aparecerá el cuadro de diálogo **Arir**.
14. Localiza el archivo _%Modelo para configurar Digi3D%\125-126.d3d,_ selecciónalo y pulsa el botón **Abrir**.
15. Pulsa el botón **Aceptar** del cuadro de diálogo **Nuevo proyecto**.
16. Se creará una ventana fotogramétrica mostrando el modelo _125-126_.
17. Maximiza la ventana fotogramétrica si no está maximizada.
18. Desmaximiza la ventana fotogramétrica mediante la opción del menú **Ventana/Mosaico vertical**.
19. Cambia el tamaño de la ventana fotogramétrica para que ocupe la posición que quieras.
