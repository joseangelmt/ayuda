# Prerequisitos

Ayuda online de productos Digi21

Si vas a instalar _Digi3D.NET_ por primera vez, primero tendrás que instalar una serie de pre-requisitos.

## Prerequisitos para instalar Digi3D.NET edición de 32 bits

1. Instala [Microsoft Visual C++ 2010 SP1 Redistributable Package](http://www.microsoft.com/es-es/download/details.aspx?id=8328).
2. Instala [Microsoft .NET Framework 4.0](http://www.microsoft.com/downloads/es-es/details.aspx?FamilyID=9cfb2d51-5ff4-4491-b0e5-b386f32c0992).

## Prerequisitos para instalar Digi3D.NET edición de 64 bits.

1. Instala [Microsoft .NET Framework 4.0](http://www.microsoft.com/downloads/es-es/details.aspx?FamilyID=9cfb2d51-5ff4-4491-b0e5-b386f32c0992).
2. Instala [Microsoft SQL Server Compact 3.5 Service Pack 2](http://www.microsoft.com/es-es/download/details.aspx?id=5783).
