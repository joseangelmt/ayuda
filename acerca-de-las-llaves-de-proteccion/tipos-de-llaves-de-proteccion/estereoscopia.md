# Estereoscopia

Esta opción del cuadro de diálogo de configuración nos permite indicar el tipo de estéreo que mostrará Digi3D cuando cargue modelos [estereoscópicos](estereoscopia.md) o [tri-estereoscópicos](estereoscopia.md).

Digi3D es capaz de mostrar los siguientes tipos de estereoscopía: 

| Page Flipping | Especifica que se mostrará estereo en función de la configuración del driver. Esta es la opción configurada por defecto. | Utiliza esta opción para configuraciones [Planar 3D](estereoscopia.md) y de estéreo activo. |
| :--- | :--- | :--- |
| Dividido en horizontal | Digi3D dividirá la ventana fotogramétrica en dos partes. En la parte izquierda mostrará la imagen a visualizar con el ojo izquierdo. En la parte derecha mostrará la imagen a visualizar con el ojo derecho. | Utiliza esta opción para utilizar con un [estereóscopo para monitores](estereoscopia.md). |
| Dividido en vertical | Digi3D dividirá la ventana fotogramétrica en dos partes. La parte superior mostrará la imagen a visualizar con el ojo izquierdo. La parte inferior mostrará la imagen a visualizar con el ojo derecho. | Este sistema está en desuso. Se utilizaba con gafas activas y tarjetas gráficas que no disponían de características de estéreo profesional. |
| Entrelazado horizontal | Digi3D mostrará en las líneas pares la imagen a visualizar con el ojo izquierdo y en las impares la imagen a visualizar con el ojo derecho. | Utiliza esta opción para trabajar con monitores de [estéreo entrelazado de gafa pasiva](estereoscopia.md). |
| Entrelazado vertical | Digi3D mostrará en las columnas pares la imagen a visualizar con el ojo izquierdo y en las impares la imagen a visualizar con el ojo derecho. | Utiliza esta opción para trabajar con [monitores auto-estereoscópicos](estereoscopia.md). |
