# Configurando el programa tras la instalación

El instalador de Digi3D.NET configura el programa para equipos con dos monitores, con una determinada configuración de estereoscopía, con un determinado dispositivo de entrada... Es posible que esta configuración no se adapte al hardware de que dispones.

En esta sección vas a aprender a configurar el programa para que se adapte a tu _hardware_ y a tus necesidades.

1. [Configuración para un único monitor o múltiples monitores](configuracion-para-un-unico-monitor-o-multiples-monitores.md).
2. [Configurando la visualización estereoscópica](configurando-la-visualizacion-estereoscopica.md).
3. [Configurando el dispositivo de entrada](configurando-el-dispositivo-de-entrada.md).
