# MDTPC

Esta orden se utiliza para generar Modelos Digitales del Terreno por Correlación.

## Parámetros

No admite parámetros

## Observaciones

Debe abrir un modelo estereoscópico para poder ejecutar esta orden en la ventana fotogramétrica.

## Características de la orden

| Tipo de orden | [Orden inmediata](mdtpc.md) |
| :--- | :--- |
| Repite automáticamente | No |
| Opción del menú donde aparece la orden | MDT/Modelo digital del terreno por correlación ... |
| Barra de herramientas en la que aparece la orden | _Esta orden no tiene asociado ningún botón en ninguna barra de herramientas_ |
| Extensión | DigiNG.OrdenesStandard.dll |
| Variables relacionadas | No tiene variables relacionadas |
