# Alquiler finalizado

Ha finalizado el período de alquiler del programa que estás ejecutando.

Existen dos soluciones a este problema:

1. Alquila el programa en nuestra [Tienda Online](http://www.digi21.net/Tienda/Compra).
2. Consolida un alquiler si ya has alquilado el programa pero aún no has almacenado las licencias en tu llave de protección con el programa [Consolidar Alquiler](consolidar-alquiler.md).
3. Cambia la llave de protección que debe utilizar el programa y selecciona una que disponga de un alquiler sin expirar o del producto comprado con el programa [Selecciona llave](selecciona-llave.md).
