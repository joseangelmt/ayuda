# Extensiones del Sensor Conico

Las extensiones del Sensor Cónico Estereoscópico permiten que éste realice las siguientes operaciones:

* Cargar [Orientacion Exterior](orientacion-exterior.md) de distintos proveedores.
* Cargar [calibraciones de cámaras](calibracion-de-camara.md) de distintos proveedores.
* Cargar archivos de proyecto en el [Panel Proyecto fotogramétrico](panel-proyecto-fotogrametrico.md).

Estas extensiones no tienen que proporcionar forzosamente los tres puntos enumerados anteriormente, sino que pueden proporcionar uno o dos de los puntos enumerados anteriormente, como por ejemplo la extensión [Digi3D.ConicSensor.Imu](digi3d.conicsensor.imu.md), que es la encargada de cargar archivos de centros de proyección y giros. Estos archivos no proporcionan ni calibraciones de cámaras ni relaciones entre modelos, de modo que esta extensión únicamente proporciona _orientaciones exteriores_ a _Digi3D.NET_.

_Digi3D.NET_ implementa las siguientes extensiones para el _Sensor Cónico_:

* [Digi3D.ConicSensor.AeroOffice](digi3d.conicsensor.aerooffice.md).
* [Digi3D.ConicSensor.Digi3D](digi3d.conicsensor.digi3d.md).
* [Digi3D.ConicSensor.Dvp](digi3d.conicsensor.dvp.md).
* [Digi3D.ConicSensor.EspaKernel](digi3d.conicsensor.espakernel.md).
* [Digi3D.ConicSensor.Imu](digi3d.conicsensor.imu.md).
* [Digi3D.ConicSensor.Inpho](digi3d.conicsensor.inpho.md).
* [Digi3D.ConicSensor.PatB](digi3d.conicsensor.patb.md).
* [Digi3D.ConicSensor.PhotoScan](digi3d.conicsensor.photoscan.md).
* [Digi3D.ConicSensor.Pix4D](digi3d.conicsensor.pix4d.md).
* [Digi3D.ConicSensor.RapidCal](digi3d.conicsensor.rapidcal.md).
* [Digi3D.ConicSensor.Summit](digi3d.conicsensor.summit.md).
