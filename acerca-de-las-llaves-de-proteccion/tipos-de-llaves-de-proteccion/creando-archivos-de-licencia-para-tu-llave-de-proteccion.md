# Creando archivos de licencia para tu llave de protección

Digi3D.NET únicamente va a ejecutarse si el ordenador tiene sus correspondientes archivos de licencia.  
Los archivos de licencia son necesarios independientemente de si la llave de protección del programa es física o de software.

Tras finalizar este tutorial el ordenador tendrá instalados los archivos de licencia necesarios para ejecutar _Digi3D.NET_.

1. Los archivos de licencia se crean para una llave de protección en particular, de modo que necesitas saber el identificador de la llave de protección para el cual quieres crear los archivos de licencia. Haz clic en este enlace: [http://localhost:1947/\_int\_/devices.html](http://localhost:1947/_int_/devices.html). Aparecerá una página web titulada _Safenet Admin Control Center_. En el contenido principal de la página verás una tabla con varias columnas: _Location, Vendor, Key ID, Key Type, Configuration, Version, Sessions_ y _Actions_.
2. En la tabla aparecerá tantas llaves de protección como se localicen tanto en tu propio ordenador como en la red de la empresa \(si es que el ordenador está conectado a la red\).
3. Localiza la llave de protección sobre la que quieres crear los archivos de licencia y copia el valor del campo _Key ID._
4. Abre el siguiente enlace: [https://www.digi21.net/Tienda/](https://www.digi21.net/Tienda/) y selecciona si quieres alquilar o comprar una actualización.
5. Localiza el producto que quieres comprar/alquilar. Introduce el _Hasp ID_ de la llave para la cual quieres crear los archivos de licencia en el campo correspondiente, selecciona la opción para indicar el número de días/meses que quieres alquilar y pulsa el botón _Comprar ahora_.
6. Realiza el pago con tu cuenta de _PayPal_ o introduciendo los datos de una tarjeta de crédito.
7. Te enviaremos por correo electrónico un enlace con el programa que creará los archivos de licencia en tu ordenador.
