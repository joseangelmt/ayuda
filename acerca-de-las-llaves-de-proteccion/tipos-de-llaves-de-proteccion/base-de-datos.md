# Base de datos

Esta pestaña muestra esquema de bases de datos. Un esquema de bases de datos consiste en una relación de tablas y de los campos de dichas tablas.

Las tablas creadas en esta pestaña se podrán seleccionar en el campo **Tabla** de la sección **Base de datos** de cada código añadido en la pestaña **Códigos**. Si abrimos con Digi3D.NET un archivo de dibujo con una tabla de códigos con esquema de base de datos y seleccionamos un código que tenga asociada una tabla, y estamos conectados con una base de datos, Digi3D.NET mostrará en el [Panel Campos de la base de datos](panel-campos-de-la-base-de-datos.md) todos los campos enumerados en el esquema.
