# Digi3D.ConicSensor.Pix4D

| Orientaciones | Calibración de cámaras | Proyectos |
| :--- | :--- | :--- |
|  |  |  |

Esta extensión hace que _Digi3D.NET_ sea compatible con archivos de calibración de cámaras creados con el programa _PatB_.

## Tipos de archivo soportados

Permite cargar _calibraciones de cámara_ a partir de las siguientes extensiones de archivos:

| Extensión | Descripción | ¿Cuantos archivos? |
| :--- | :--- | :--- |
| _\_pix4d\_calibrated\_internal\_camera\_parameters.cam_ | Archivo con parámetros de orientación externa y calibración de cámaras. | Tantos como cámaras distintas se hayan utilizado en el proyecto. |
