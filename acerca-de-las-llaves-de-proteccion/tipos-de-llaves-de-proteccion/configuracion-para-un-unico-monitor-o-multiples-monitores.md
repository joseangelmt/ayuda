# Configuración para un único monitor o múltiples monitores

El instalador de Digi3D.NET asume que tu ordenador dispone de dos monitores y que utilizarás uno para visualizar estereoscopía y el otro para la ventana de dibujo.

Si no cambias esta configuración, al abrir una ventana fotogramétrica, ésta se creará como una aplicación independiente que puedes arrastrar a cualquier monitor. Esta ventana podrá salir de los límites de la ventana principal de la aplicación.

Si tu ordenador dispone de un único monitor, cuando maximices esta ventana, se **ocultará** la ventana principal de la aplicación, de modo que no podrás ver la ventana de dibujo haciendo que sea muy complicado trabajar.

Sigue los pasos de [Configurando el programa para un único monitor](configurando-el-programa-para-un-unico-monitor.md) para aprender a solucionar este problema.

Si añades un segundo monitor a tu ordenador y tienes configurado Digi3D.NET para un único monitor y deseas configurarlo para varios monitores, sique los pasos de [Configurando el programa para varios monitores](configurando-el-programa-para-varios-monitores.md) para cambiar esa configuración.
