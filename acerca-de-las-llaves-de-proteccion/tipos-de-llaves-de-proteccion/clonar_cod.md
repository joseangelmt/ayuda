# CLONAR\_COD

Establece como códigos activos los de la entidad seleccionada.

## Parámetros

No admite parámetros.

## Observaciones

Además de establecer como códigos activos los códigos de la entidad seleccionada, asigna los siguientes valores a las siguientes variables:

| Variable | Valor |
| :--- | :--- |
| [BLOQUEA\_Z](bloquea_z.md) | Falso |
| [C](c.md) | Falso |
| [FIJA\_Z](FIJA_Z.html) | Falso |
| [FORMATO\_AUTONUM](formato_autonum.md) | %d |
| [IR\_PRINCIPIO](ir_principio.md) | 0 |
| [MAXPUNTOS](maxpuntos.md) | 0 |
| [ORTO](orto.md) | Falso |
| [ORTO\_AA](orto_aa.md) | Falso |
| [S](s.md) | Falso |
| [TIPO\_DE\_Z](tipo_de_z.md) | Falso |
