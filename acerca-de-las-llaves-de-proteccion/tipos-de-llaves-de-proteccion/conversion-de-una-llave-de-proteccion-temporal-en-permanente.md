# Conversión de una llave de protección temporal en permanente

Las llaves de protección temporales tienen una duración máxima de 30 días.

Una vez transcurridos 30 días la llave de protección temporal deja de funcionar y hay que convertirla en una llave de protección permanente.

Sigue las siguientes instrucciones para convertir tu llave de protección temporal en una llave de protección permanente:

1. Convertir tu llave de protección temporal en una permanente tiene un precio de 20€. Haz clic en este enlace para realizar el pago: [Comprar ahora](https://www.paypal.com/cgi-bin/webscr?cmd=_s-xclick&hosted_button_id=G6Q4XR96MFKLU).
2. Descarga ejecuta el programa [ActualizarLlaveLDK](http://digi21.blob.core.windows.net/download/ActualizarLlaveLDK.exe) Este programa sirve para analizar tu llave de protección y crear un archivo que nos va a permitir reprogramarla remotamente.
3. Aparecerá el programa _Sentinel HASP RUS_. Este programa tiene dos pestañas: _Collect Key Status Information_ y _Apply License Update_.
4. En la pestaña _Collect Key Status Information_ pulsa el botón _Apply License Update._
5. Aparece el cuadro de diálogo _Save Key Status As_. Este cuadro de diálogo te va a permitir indicar la ubicación y nombre del archivo [.c2v](conversion-de-una-llave-de-proteccion-temporal-en-permanente.md) que nos vas a enviar.
6. Indica la ubicación y nombre del archivo _.c2v_ a crear.
7. Envíanos el archivo que acabas de crear a la la dirección de correo electrónico: [info@digi21.net](mailto:info@digi21.net) junto con la siguiente información:
   * Nombre de la empresa
   * N.I.F.
   * Dirección
   * Código postal
   * Persona de contacto
   * Teléfono
   * Correo electrónico
8. Una vez verificado que has realizado el pago de la llave, te responderemos con un archivo con extensión [.v2c](conversion-de-una-llave-de-proteccion-temporal-en-permanente.md) que vas a utilizar para programar tu llave de protección.
9. Haz clic en el siguiente enlace: [http://localhost:1947/\_int\_/checkin.html](http://localhost:1947/_int_/checkin.html). Aparecerála página web titulada _Safenet Admin Control Center._
10. Pulsa el botón _Examinar._
11. Aparecerá el cuadro de diálogo titulado _Elegir archivos para cargar_.
12. Selecciona el archivo _.v2c_ que te habremos enviado por correo electrónico y pulsa el botón _Abrir_.
13. Pulsa el botón _Apply File_.

El número de serie de tu llave de protección habrá cambiado, de modo que tendrás que volver a [crear los archivos de licencia](creando-archivos-de-licencia-para-tu-llave-de-proteccion.md) para la llave.
