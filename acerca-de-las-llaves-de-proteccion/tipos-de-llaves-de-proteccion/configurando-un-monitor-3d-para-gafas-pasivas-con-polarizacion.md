# Configurando un monitor 3D para gafas Pasivas con Polarización

Estos monitores utilizan gafas pasivas y no requieren de un emisor ni de una tarjeta gráfica profesional. Tienen el inconveniente de que al ser polarizados no vemos el 100% de las líneas del monitor con cada ojo. Cada ojo visualizará únicamente la mitad de líneas del monitor, viendo siempre líneas negras intercaladas.

No es necesario configurar el controlador de la tarjeta gráfica para este tipo de monitor. Tan solo tendremos que configurar Digi3D.NET.

Sigue los siguientes pasos para configurar Digi3D.NET para visualizar estereoscopía en un monitor _3D_:

1. Ejecuta **Digi3D.NET**.
2. Aparecerá el cuadro de diálogo **Nuevo proyecto**.
3. Pulsa el botón **Cancelar** para cerrar el cuadro de diálogo **Nuevo proyecto**.
4. Selecciona la opción del menú **Herramientas/Configuración**.
5. Despliega la sección **Ventana fotogramétrica**.
6. Selecciona la opción **Entrelazado horizontal** en campo **Estereoscopía**.
7. Pulsa el botón **Aceptar** del cuadro de diálogo **Configuración**.
