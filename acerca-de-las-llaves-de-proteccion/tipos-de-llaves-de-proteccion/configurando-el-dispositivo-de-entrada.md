Configurando el dispositivo de entrada

La ventana fotogramétrica se pude controlar con el ratón del sistema o con un dispositivo específico como un _TopoMouse_, un _SoftMouse_, un ratón _Stealth 3D Mouse_ o con _manivelas_.

En esta sección vamos a aprender a configurar Digi3D.NET para que funcione con tu dispositivo de entrada.

* Configura Digi3D.NET para trabajar con tu [TopoMouse](configura-digi3d.net-para-trabajar-con-un-topo-mouse.md).
* Configura Digi3D.NET para trabajar con tu [SoftMouse](configura-digi3d.net-para-trabajar-con-un-soft-mouse.md).
