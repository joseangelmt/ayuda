# Configurando Digi 3D para gafas de Anaglifo

Si no dispones de un monitor 3D, puedes visualizar estereoscopía mediante gafas de anaglifo.

No es necesario configurar el controlador de la tarjeta gráfica para este tipo de monitor. Tan solo tendremos que configurar Digi3D.NET.

Sigue los siguientes pasos para configurar Digi3D.NET para visualizar estereoscopía con gafas de anaglifo:

1. Ejecuta **Digi3D.NET**.
2. Aparecerá el cuadro de diálogo **Nuevo proyecto**.
3. Pulsa el botón **Cancelar** para cerrar el cuadro de diálogo **Nuevo proyecto**.
4. Selecciona la opción del menú **Herramientas/Configuración**.
5. Despliega la sección **Ventana fotogramétrica**.
6. Selecciona la opción **Anaglifo** en campo **Estereoscopía**.
7. Pulsa el botón **Aceptar** del cuadro de diálogo **Configuración**.

Este sistema tiene problemas con el color porque ciertos colores desaparecen, por eso en ocasiones es conveniente mostrar las imágenes en tonos de gris aunque sean en color. Puedes solucionar este inconveniente visualizando en la ventana fotogramétrica las imágenes en tonos de gris aunque estas sean en color.

1. Con un modelo fotogramétrico cargado, selecciona la opción del menú **Ventana fotogramétrica/Propiedades de visualización**. Aparecerá el [Panel Propiedades de visualización](panel-propiedades-de-visualizacion.md).
2. Activa la casilla **Tonos de gris**.
3. Cierra el panel **Propiedades de visulización**.
