# Barra de herramientas Zooms

Conoce la Barra de herramientas **Zooms**:

| Botón | Descripción | Orden |
| :--- | :--- | :--- |
| Regenera | Permite regenerar la pantalla | [REGENERA](regenera.md) |
| Zoom extendido | Muestra en pantalla todo el archivo de dibujo | [ZOOME](zoome.md) |
| Zoom ventana | Muestra una parte del dibujo mediante una ventana generada por el usuario | [ZOOMV](zoomv.md) |
| Zoom de acercar | Nos acerca con respecto al centro de la ventana de dibujo | [ZOOM+](zoom+.md) |
| Zoom de alejar | Nos aleja con respecto al centro de la ventana de dibujo | [ZOOM-](zoom.md) |
| Zoom a entidad | Realiza un zoom a una entidad seleccionada por el usuario | [ZOOM\_ENTIDAD](zoom_entidad.md) |

El menú Zooms, muestra estas opciones además de zooms avanzados que aprenderás a utilizar más adelante.
