# ENVIA\_DATOS\_DRIVER

Envía datos a los drivers programados para la orden [DISPOSITIVO\_ENTRADA](dispositivo_entrada.md).

## Parámetros

Esta orden no admite parámetros.

## Observaciones

No se han registroado observaciones para esta orden.

## Características de la orden

| Tipo de orden | [Orden inmediata](envia_datos_driver.md) |
| :--- | :--- |
| Repite automáticamente | No |
| Opción del menú donde aparece la orden | _Esta orden no tiene asociada ninguna opción de menú_ |
| Barra de herramientas en la que aparece la orden | _Esta orden no tiene asociado ningún botón en ninguna barra de herramientas_ |
| Extensión | DigiNG.OrdenesStandard.dll |
| Variables relacionadas | No tiene variables relacionadas |
