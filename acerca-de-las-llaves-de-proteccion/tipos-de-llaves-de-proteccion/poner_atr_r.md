# PONER\_ATR\_R

Al ejecutar esta orden Digi3D.NET pedirá que selecciones las entidades a las que deseas asignar los atributos.

## Parámetros

Los únicos parámetros que va a tener en cuenta esta orden serán los atributos que desees asignar a las entidades.

## Observaciones

En el caso de hacer click sobre una zona en la que no hubiese recinto, Digi3D.NET mostrará una señal acústica de error para informar al usuario.

## Características de la orden

| Tipo de orden | [Orden interactiva](poner_atr_r.md) |
| :--- | :--- |
| Repite automáticamente | No |
| Opción del menú donde aparece la orden | _Esta orden no tiene asociada ninguna opción de menú_ |
| Barra de herramientas en la que aparece la orden | _Esta orden no tiene asociado ningún botón en ninguna barra de herramientas_ |
| Extensión | DigiNG.OrdenesStandard.dll |
| Variables relacionadas | No tiene variables relacionadas |
