# TEXTO\_R

Inserta un texto en el dibujo en la dirección indicada por el usuario.

## Parámetros

| Número de parámetro | Descripción | Valores | Opcional |
| :--- | :--- | :--- | :--- |
| 1 | Código texto \([COD](cod.md)\) | Identificador del código | Si |
| 2 | Altura de texto \([AT](at.md)\) | Número real | Si |
| 3 | Justificación de texto \([JT](jt.md)\) | Número real | Si |

## Observaciones

No se han registrado observaciones para esta orden.

## Características de la orden

| Tipo de orden | [Orden interactiva](texto_r.md) |
| :--- | :--- |
| Repite automáticamente | Si |
| Opción del menú donde aparece la orden | Dibujar/Texto con dos puntos |
| Barra de herramientas en la que aparece la orden | Textos |
| Extensión | DigiNG.OrdenesRaster.dll |
| Variables relacionadas | No tiene variables relacionadas |
| Variables relacionadas | [AT](at.md), [JT](jt.md), [REPITE](repite.md) |
