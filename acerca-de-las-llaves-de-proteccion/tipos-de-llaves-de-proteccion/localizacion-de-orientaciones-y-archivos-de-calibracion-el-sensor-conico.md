# Localización de orientaciones y archivos de calibración el sensor cónico

Tal y como se ha explicado en [Lógica para transformar de coordenadas terreno a pixel](logica-para-transformar-de-coordenadas-terreno-a-pixel.md), el sensor cónico estereoscópico necesita para realizar su lógica una [Orientación Absoluta](orientacion-absoluta.md), una [Orientacion Exterior](orientacion-exterior.md) para cada imagen y un archivo de [Calibracion de cámara](calibracion-de-camara.md) para cada imagen. Cuando cargamos un modelo con este sensor, éste intenta localizar estas orientaciones siguiendo los siguientes pasos:

* [Localización del archivo de orientación absoluta](localizacion-del-archivo-de-orientacion-absoluta.md).
* [Localización de orientaciones](localizacion-de-orientaciones.md).
* [Localización del archivo de calibración de cámara](Localizaci-n%20del%20archivo%20de%20calibraci-n%20de%20c-mara.html).
