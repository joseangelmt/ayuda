# Importar símbolos de un archivo .cel de MicroStation

Esta herramienta, accesible a través del menú **Símbolos/Importar símbolos de un archivo** [**.cel**](importar-simbolos-de-un-archivo-.cel-de-microstation.md) **de MicroStation** permite crear en un directorio especificado tantos archivos [.bin](importar-simbolos-de-un-archivo-.cel-de-microstation.md) como símbolos se localicen en el archivo de células especificado.

Al seleccionar la opción del menú, se muestra un cuadro de diálogo titulado **Importar símbolos de un archivo .cel** en el que se pueden seleccionar el archivo .cel a importar así como el directorio de salida en el que almacenar los archivos .bin generados.

Al pulsar el botón **Importar** se importará el archivo .cel y se generarán tantos archivos .bin como células se localicen en el archivo .cel.
