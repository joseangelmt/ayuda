# Transformador Universal De Coordenadas

El programa Transformador Universal de Coordenadas permite transformar coordenadas de un [sistema de referencia de coordenadas](sistemas-de-referencia-de-coordenadas.md) origen a un sistema de coordenadas destino.

Su uso es muy sencillo, tan solo tendremos que seleccionar el sistema de referencia de coordenadas origen, el de destino, y pegar las coordenadas a transformar en la parte inferior izquierda del programa. El programa mostrará inmediatamente la transformación en la parte inferior derecha.

Este programa únicamente transforma entre sistemas de referencia de coordenadas de referencia 3D, de modo que no podremos seleccionar un sistema de referencia de coordenadas horizontal o vertical, siempre nos va a forzar a seleccionar uno [geográfico 3D](sistemas-de-referencia-de-coordenadas-geograficos-3d.md), o uno [compuesto](sistemas-de-referencia-de-coordenadas-compuestos.md), formado por uno _geográfico 2D + vertical_ o un _geográfico 3D + vertical_.

Puedes descargar el código fuente de este programa de la siguiente dirección: [https://github.com/digi21/TransformadorUniversalCoordenadas](https://github.com/digi21/TransformadorUniversalCoordenadas).
