# Barra de herramientas Selecciones

Este es el aspecto de la Barra de herramientas **Selecciones**:

| Icono | Nombre | Descripción |
| :--- | :--- | :--- |
|  | Primer botón | Selecciona todo |
|  | Segundo botón | Selecciona múltiples entidades |
|  | Tercer botón | Selecciona todas las entidades que tengan entre sus códigos alguno de los seleccionados |
|  | Cuarto botón | Selecciona línea |
|  | Quinto botón | Seleccionar ventana por solape |
|  | Sexto botón | Seleccionar ventana por inclusión |
|  | Séptimo botón | Seleccionar entidades exteriores a la ventana |
|  | Octavo botón | Seleccionar polígono por solape |
|  | Noveno botón | Seleccionar polígono por inclusión |
|  | Décimo botón | Seleccionar entidades exteriores al polígono |
|  | Onceavo botón | Seleccionar la última entidad del archivo de dibujo/Selecciona última entidad |
|  | Doceavo botón | Selecciona posición |
|  | Treceavo botón | Selecciona marcados |
